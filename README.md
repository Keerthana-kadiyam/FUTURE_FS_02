# Client Lead Management System (Mini CRM)
This is a simple web application to manage and track client leads.
## Features
- Add, edit, and delete leads  
- Track lead status (New, Contacted, Converted, Lost)  
- Dashboard showing total, converted, and pending leads  
- Follow-up tracking  
- Clean and responsive design  
## Tech Used
- HTML  
- CSS  
- JavaScript
